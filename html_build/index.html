주요 기능 정의서

1. 사전예약 메인 기능
  • GNJOY 로그인 기반 참여 시 약관 동의 (개인정보 수집 및 이용) 체크박스 플로우 진행
  • 비로그인 시: 로그인 유도 화면 노출
  • 로그인 후 (사전예약 전 상태): 사전예약 신청 버튼 노출
  • 로그인 후 (사전예약 완료 상태): 사전예약 완료 노출
2. 글로벌 마일스톤 (달성도) 기능
  • 글로벌 사전예약 누적 인원 수에 비례하여 게이지바가 실시간으로 갱신됩니다.
  • 지정된 인원수(예: 10만, 50만, 100만)를 돌파할 때마다 특정 보상이 해금(Unlock)됩니다.
  • 달성 여부에 따라 보상 아이콘의 활성화(컬러) 및 비활성화(흑백) 상태가 시각적으로 표시됩니다.
3. ROZ 크로스 이벤트 기능
  • GetPoring과 라그나로크 제로(ROZ) 두 게임 모두 사전예약 시 추가 보상을 지급합니다.
  • 페이지 내에서 두 게임의 사전예약 참여 여부를 각각의 카드로 실시간 연동해 보여줍니다.
  • 양측 이벤트가 모두 완료 상태가 되었을 때만 하단의 보상 수령 버튼이 활성화됩니다.
4. 데일리 출석체크 기능
  • 이벤트 기간 동안 매일 사이트에 방문하여 로그인 기반으로 출석체크를 진행합니다.
  • 로그인 버튼 클릭 시 당일 출석이 인정되며 타임존 정책에 따라 일1회 출첵 가능합니다.
  • 누적 출석 일수에 따라 단계별로 GetPoring 전용 특별 보상을 획득할 수 있습니다.
