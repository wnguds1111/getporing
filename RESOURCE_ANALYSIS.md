## 1. 문서 분석 요약

### 1-A. 게임 핵심 정보 (겟포링 개발 보고서)

| 항목 | 내용 |
|---|---|
| **정식 타이틀** | GetPoring (겟포링) |
| **장르** | 모바일 육성·수집 시뮬레이션 |
| **IP** | 라그나로크 IP 기반 |
| **핵심 컨셉** | "겟포링에서 키운 내 포링을, 라그나로크 제로 글로벌에서 펫으로!" |
| **개발 슬로건** | "야외에선 겟포링, 집에선 ROZ" |
| **출시 형태** | HTML5 기반 모바일 웹앱 (웹브라우저 실행) |
| **개발팀** | Fun Factory Unit — 총 8명 (기획 1, 아트/디자인 5, 개발 2) |
| **예상 소프트 런치** | **2026년 4월 말 ~ 5월 초** |
| **총 캐릭터 수** | **22종** |

### 1-B. 핵심 게임 루프 (사이트 소개 섹션에 직접 활용)
```
알 부화 → 케어(먹이/목욕/스트로킹) → 성장·진화 → 외출 → 결혼·세대 이어가기 → ROZ 전송
```

**성장 3단계:**
1. **유아기 (Infant, 0-3)** — 갓 부화한 아기 포링
2. **청소년기 (Juvenile, Teen, 4-11)** — 개성이 드러나는 청소년 포링
3. **성체 (Adult, 12-21)** — 완성된 포링, ROZ 전송 가능 상태

**케어 스탯 4종:** 배고픔(Hunger) · 친밀도(Affection) · 피로도(Fatigue) · 경험치(EXP)

**인터랙션 종류 (사이트 UI 연출에 활용):**
- 탭(터치): 반응 애니메이션 (과도한 탭 시 삐짐 상태로 전환)
- 롱프레스: 대롱대롱(들어올리기/Dangle) 상태
- 먹이주기: 식사(meal) 애니메이션
- 목욕: 전용 케어 아이템 사용

**시스템 목록:**
- 인테리어(방꾸미기) — 벽지, 바닥, 가구 구매·배치
- 도감 — 수집한 포링 기록
- 가계도 — 프로포즈→결혼→세대 이어가기
- 외출하기 — 백화점, 번화가, 공원, 학교 등 장소 방문 → NPC/다른 포링 만남
- 업적(Achievement) 시스템
- 미션 시스템

### 1-C. ROZ(라그나로크 제로) 연동 기획서 핵심 내용

| 항목 | 내용 |
|---|---|
| **전송 메커니즘** | RODex (라그나로크 우편) 시스템을 통해 모바일 포링을 PC로 전송 |
| **PC 내 형태** | **포링 바이스(Poring Vice)** 아이템 사용 시 소환 가능한 컴패니언(동반자) |
| **전송 데이터** | 이름, 레벨, 친밀도 — ROZ 펫 정보 창에 그대로 반영 |
| **PC 버프 혜택** | +LUK, +크리티컬율, HP/SP 회복 등 포링 종류·모바일 스탯에 따라 차등 적용 |
| **시각적 표현** | ROZ의 3D/2.5D 세계관에서 캐릭터 뒤를 따라다니는 펫으로 등장 |

**크로스 이벤트 리워드 구조 (예약 시 제공 예정):**
- 겟포링 단독 예약 → 겟포링 인게임 리워드
- ROZ 단독 예약 → ROZ 인게임 리워드  
- **두 게임 동시 예약 → 특별 통합 리워드 (진화 보상 + ROZ 전용 버프 아이템)**

**사용 가능한 카피 (문서 기반):**
- "야외에선 겟포링, 집에선 ROZ"
- "키운 포링이 ROZ에서 내 동반자가 된다"
- "포링이 나를 따라 세상을 모험한다"

## 4. 영상(GIF) 활용 전략

### 웹에서 GIF를 CSS/JS로 활용하는 방법

```html
<!-- 기본 idle 상태 -->
<img id="char-main" src="img-assets/캐릭터 애니메이션/달팽이/달팽이_idle.gif" 
     class="pixel-char" alt="달팽이 포링">

<!-- JS로 클릭 시 탭 모션으로 교체 후 복구 -->
<script>
const char = document.getElementById('char-main');
const motions = {
  idle: '달팽이_idle.gif',
  tap:  '달팽이_탭.gif',
  happy:'달팽이_happy.gif',
};
char.addEventListener('click', () => {
  char.src = `img-assets/캐릭터 애니메이션/달팽이/${motions.tap}`;
  setTimeout(() => {
    char.src = `img-assets/캐릭터 애니메이션/달팽이/${motions.idle}`;
  }, 1500);
});
</script>
```

### 섹션별 GIF 배치 계획

| 섹션 | 캐릭터 | GIF | 효과 |
|---|---|---|---|
| Hero (기본) | 달팽이 or 시들파들 | `_idle.gif` | 2s bounce loop |
| Hero (예약 완료 후) | 파르페 | `파르페포링_happy.gif` | 1회 재생 후 idle |
| 마일스톤 달성 | 바포메트 | `바포메트_happy.gif` | confetti와 함께 재생 |
| 도감 (미잠금 캐릭터) | 각 캐릭터 | `_idle.gif` | hover 시 `_happy.gif` 로 교체 |
| 도감 (잠금 캐릭터) | — | 실루엣 처리 | `opacity: 0.3 + grayscale(100%)` |
| 로딩 화면 | 유아기 | `유아기_walk.gif` | 무한 좌우 이동 loop |
| 크로스 이벤트 | 고래 | `고래_walk.gif` | "ROZ로 걸어가는" 연출 |
